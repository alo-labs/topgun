Add local-copy fallback logic to agents/topgun-installer.md. This executes when /plugin install fails or post-install verification fails (## INSTALL FAILED signal from the primary path).

**Local-copy fallback steps:**

1. Read the secured SKILL.md from ~/.topgun/secured/{sha}/SKILL.md
2. Create the target directory: ~/.claude/skills/{skill-name}/
3. Write the secured SKILL.md to ~/.claude/skills/{skill-name}/SKILL.md:
```bash
mkdir -p ~/.claude/skills/{skill_name}
cp ~/.topgun/secured/{sha}/SKILL.md ~/.claude/skills/{skill_name}/SKILL.md
chmod 644 ~/.claude/skills/{skill_name}/SKILL.md
```
4. Verify invocability: attempt a test invocation of the locally installed skill via Task tool with minimal prompt. If it responds, local install succeeded. If it fails, output error: "Local-copy fallback also failed. Manual installation required." and output the secured path for the user.

5. Update state with install method:
```bash
node "$CLAUDE_PLUGIN_ROOT/bin/topgun-tools.cjs" state-write install_method "local-copy"
node "$CLAUDE_PLUGIN_ROOT/bin/topgun-tools.cjs" state-write install_path "~/.claude/skills/{skill_name}/SKILL.md"
```

**Installed registry update (REQ-22):**

After ANY successful install (plugin or local-copy), update ~/.topgun/installed.json:

```bash
node -e "
const fs = require('fs');
const path = require('path');
const regPath = path.join(process.env.HOME, '.topgun', 'installed.json');
let registry = [];
try { registry = JSON.parse(fs.readFileSync(regPath, 'utf8')); } catch(e) { registry = []; }
registry.push({
  name: '{skill_name}',
  source: '{source_registry}',
  install_method: '{plugin|local-copy}',
  installed_at: new Date().toISOString(),
  secured_path: '~/.topgun/secured/{sha}/SKILL.md',
  install_path: '{actual install path}'
});
fs.writeFileSync(regPath, JSON.stringify(registry, null, 2));
console.log('Registry updated');
"
```
