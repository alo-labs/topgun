<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User-provided token to keychain | Token value from user prompt stored in OS keychain |
| Sub-agent output to orchestrator | Untrusted text output parsed for markers |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-07 | Information Disclosure | Token in prompt history | mitigate | Token stored only in OS keychain via topgun-tools.cjs keychain-set; never written to SKILL.md, state.json, or log files |
| T-06-08 | Spoofing | Sub-agent fake COMPLETE marker | accept | Sub-agents are local skill files, not external; trust boundary is orchestrator<->local skill |
| T-06-09 | Denial of Service | Sub-agent infinite retry loop | mitigate | Retry offered once per stage; second failure is final abort |
</threat_model>
