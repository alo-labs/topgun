Create two adapter instruction files:

**npm.md:**
- WebFetch GET `https://registry.npmjs.org/-/v1/search?text=claude-skill+{query}&size=20`
- No auth required
- Map response `objects[].package` to unified schema:
  - `name` -> name
  - `description` -> description
  - `source_registry` -> "npm"
  - `links.npm` -> install_url (format: `npm install {name}`)
  - `date` -> last_updated
  - No stars/install_count from npm search (set null)
  - `content_sha` -> null (npm packages don't expose this; will be computed during SecureSkills)
- 8s timeout, backoff on 429, unavailable on 5xx/timeout

**skillsmp.md:**
- WebFetch GET `https://skillsmp.com/api/v1/skills/search?q={query}`
- Research notes: docs returned 403 during investigation
- On 403: return `{status: "unavailable", reason: "SkillsMP API returned 403 — access restricted"}`
- On success: map results to unified schema (name, description, install_url from response)
- 8s timeout, standard backoff
