- Map to unified schema fields

**agentskill-sh.md:**
- Bash: `ags search "{query}" --json` (8s timeout)
- Parse JSON output for name, description, contentSha (from ecosystem field), install_url
- If `ags` not installed, return `status: "unavailable"`, reason: "ags CLI not found"

**smithery.md:**
- Retrieve Bearer token: `node "$CLAUDE_PLUGIN_ROOT/bin/topgun-tools.cjs" keychain-get smithery_token`
- If no token: proceed without auth (lower rate limit)
- WebFetch GET `https://registry.smithery.ai/servers?q={query}` with Authorization header if token exists
- Map server entries to unified schema

**github.md:**
- WebFetch GET `https://api.github.com/search/repositories?q={query}+topic:claude-skill&sort=stars&per_page=20`
- Optional: retrieve GitHub token via `topgun-tools.cjs keychain-get github_token` for higher rate limit
- Map: full_name -> name, description, stargazers_count -> stars, html_url -> install_url, pushed_at -> last_updated

**gitlab.md:**
- WebFetch GET `https://gitlab.com/api/v4/projects?search={query}+claude-skill&order_by=stars&per_page=20`
- Map: path_with_namespace -> name, description, star_count -> stars, web_url -> install_url, last_activity_at -> last_updated
